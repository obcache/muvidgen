For developers: How license verification works

License Generation Process
$ sign-license.js generate-keypair --out license-keypair.json $ sign-license.js sign --private license-keypair.json --payload payload.json --out license.txt
Verification

The application verifies licenses offline using the embedded public key (LICENSE_PUBLIC_KEY_JWK). The private key remains secure on your server and is never distributed.
